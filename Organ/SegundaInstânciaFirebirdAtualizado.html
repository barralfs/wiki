Configuração da Porta

Passo 1: Identificamos a pasta de instalação do firebird (1).

C:\Program Files\Firebird\Firebird_3_0

Passo 2: Encontramos o arquivo 'firebird.conf' (1) e abrimos ele com o bloco de notas (2). Realizamos uma busca (3) e encontramos a #RemoteServicePort = 3050(4).

Passo 3: Removemos a '#' e depois trocamos a porta para 3052.

Final: Salvamos o bloco de notas com os campos editados conforme a imagem acima.

Configuração do Serviço

Passo 1: Executar o CMD e inserir o comando abaixo.

cd C:\Program Files\Firebird\Firebird_3_0

Passo 2: Inserir o comando de instalação do firebird.

instreg install

Passo 3: Inserir o comando de instalação do serviço.

instsvc install -auto -name Firebird_Frontsys

Passo 4: Inserir o comando de execução do serviço.

instsvc start -name Firebird_Frontsys

Final: fechar a tela de CMD.
