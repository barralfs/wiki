Estrutura do Código de Barras

O código possui 13 dígitos e pode ser dividido em 3 partes.

Parte 1: o país ao qual o produto se refere.

Parte 2: o código do produto.

Parte 3: a quantidade do produto ou o seu preço total.

Parte 4: dígito verificador, este é gerado automaticamente.

Compreendendo a 3ª parte do código de barras.

Existem dois tipos de comportamento de leitura para a terceira parte do código. Um informa a quantidade do produto e o outro informa o seu preço total.

Código com Quantidade do Produto

O sistema interpreta o código de barras dessa forma para o caso do produto cadastrado ser considerado 'Pesável'.

É possível parametrizar para que todos os códigos sejam lidos como quantidade. Para isso é necessário marcar o parâmetro 'Ler Quantidade de Produtos Unitários', dentro do cadastro de Layout de Código de Barras.

Pela configuração acima teremos a segunda parte informando o código do produto, começando na casa 2 e com 4 casas de tamanho. A terceira parte do código informa a quantidade começando na casa 8, com 5 casas de tamanho e com 3 casas decimais.

Código do produto: 9012

Quantidade: 12,345

Com a informação do código do produto o sistema conseguirá buscar o seu valor unitário e, assim, realizar um cálculo levando em conta a quantidade lida para encontrar o preço total dessa venda.

Código com Preço do Produto

Quando o produto não é assinalado como pesável o sistema irá interpretar a terceira parte do código de barras como seu preço total. Também é possível parametrizar para que os códigos de barras sejam lidos apenas dessa maneira, assinalando a opção 'Ler Preço de Produtos Pesáveis'.

Pela configuração acima teremos a segunda parte informando o código do produto, começando na casa 2 e com 4 casas de tamanho. A terceira parte do código informa o preço total começando na casa 8, com 5 casas de tamanho e com 2 casas decimais.

Código do produto: 9012

Preço: R$ 123,45

Com a informação do código do produto, o sistema conseguirá buscar o seu preço unitário e, assim, realizar realizar um cálculo levando em conta o preço lido para encontrar a quantidade total dessa venda.
