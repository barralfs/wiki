Autor Original: Lucas Coutinho

Passo 1: Realizar cópia dos arquivos "Instalar.exe" e "ServiceIntegradorOrganStats" presentes neste caminho:

"Nextcloud\FrontsysSistemas\Instalar\IntegradorOrganStats"

Passo 2: Colar na pasta raiz do Organ. Lembre-se de deixar a pasta aberta, pois, ainda precisaremos de informações do arquivo "Config".

 

Parametrizar o Arquivo Config

Passo 1: Pegar as informações do 'Config' do IntegradorOrganStats.

 

Passo 2: Copiar o texto do Integrador e jogar no 'Config' Organ. É necessário manter a ordem dos campos.

 

Criar Cliente no AppControl

Passo 1: Acessar o site para o cadastro.

"https://frontsys.com.br/util/appcontrol/"
 

Passo 2: Entrar na aba 'Clientes'.

 

Passo 3: Verifique se o cliente já possui um cadastro existente e, se não for o caso, vamos criar um novo cadastro.
 

 

Passo 4: Certifique-se de que o cliente esteja com status "ATIVO" e, depois, será necessário pegar as seguintes informações: ID do Cliente, CNPJ e Chave.

 

Passo 5: No 'Config' do Organ adicionaremos as informações reunidas no passo anterior.

 

Processo de Instalação

Passo 1: Execute o CMD como administrador e em seguida digite:

cd C:\Organ (Pasta Raiz do Organ)
(Enter)
ServiceIntegradorOrganStats /Install
(Enter)

 

 

Iniciando Serviço

Abrir Menu Iniciar > Serviços > IntegradorOrganStats > Iniciar.

 

Verificar Envio de Dados

Passo 1: Na pasta raiz do Organ, será criado um 'Log'. Ao abrirmos ela, encontraremos um arquivo de texto que irá nos dizer se os dados estão sendo enviados.

 

Passo 2: Verificar se o envio foi realizado com sucesso, lembrando que devemos esperar 1 minuto para que o arquivo seja atualizado.

 

Configurar Tempo de Atualização de Dados

No 'Config' do Organ, vamos alterar o tempo de envio das informações para o aplicativo. Deixamos o 'IntervaloMinutos' como 15, para garantir que o processo não sobrecarregue o sistema.

 

O passo final é reiniciar o serviço do Integrador e a instalação estará completa.
