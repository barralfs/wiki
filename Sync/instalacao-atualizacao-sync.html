Instalação e atualização do Sync

Este artigo descreve os procedimentos para instalação do Sync em uma nova máquina e atualização do Sync em ambientes já existentes.

Instalação do Sync em nova máquina

  • Obtenção da pasta do Sync
    • Se o cliente já utiliza o Sync:
      1. Acesse uma máquina onde o Sync já esteja instalado.
      2. Copie a pasta C:\Sync.
      3. Dica: compacte a pasta para facilitar o envio.
    • Se o cliente ainda não utiliza o Sync:
      1. Copie a pasta padrão disponível em: F:\Todos\Instalar\Sync
  • Preparação da máquina de destino
    1. Na máquina onde o Sync será instalado, cole ou extraia a pasta C:\Sync.
    2. Acesse a pasta: C:\Sync\Util
    3. Execute o instalador Microsoft ODBC Driver 18 for SQL Server.
  • Criação do atalho do Sync
    1. Na pasta C:\Sync, localize o arquivo AutoUpdate.exe.
    2. Crie um atalho na área de trabalho.
    3. Renomeie o atalho para Sync.
  • Licenciamento do Sync
    1. Abra o Sync pelo atalho criado.
    2. Acesse o menu Licenciamento.
    3. Copie o Identificador do computador exibido.
  • No FrontDesk:
    1. Acesse Clientes.
    2. Localize a empresa.
    3. Clique com o botão direito sobre a empresa e selecione Computadores Licenciados.
  • Clique em Inserir e preencha os campos solicitados:
    1. Em Descrição, digite o nome do usuário ou da máquina fornecido pelo cliente.
    2. Em Sistema, selecione Sync.
    3. Cole o Identificador do computador e clique em Gerar.
    4. Preencha o campo Expiração utilizando a mesma data dos demais computadores do cliente, vista na tela anterior.
    5. Mantenha a opção Forçar expiração desmarcada.
  1. Clicar em Inserir, preencher os campos solicitados, colar o identificador e clicar em Gerar.
  2. Preencher o campo Expiração utilizando a mesma data dos demais computadores do cliente
  3. Manter a opção Forçar expiração desmarcada e salvar
  4. Retornar ao Sync, clicar em Buscar Licença e depois em Registrar
  5. Realizar o teste de acesso utilizando o usuário ADMIN

Atualização do Sync

  1. Acessar a pasta C:\Sync na máquina do cliente.
  2. Renomear o arquivo Sync.exe para Sync_old_[data] .exe.
  3. No seu ambiente, acessar: Nextcloud\FrontsysSistemas\Instalar\Sync, copiar o arquivo Sync.exe e colá-lo na pasta C:\Sync do cliente.
  4. Executar AutoUpdate.exe para validar a atualização.
