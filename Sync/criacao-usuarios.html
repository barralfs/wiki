Criação de usuários

Na tela inicial do sistema, clique em Usuários.

Clique em Inserir para iniciar o cadastro.

Na tela de usuários:

  • Caso o cliente utilize o RM, selecione o Usuário Associado ao ERP.
  • Caso não utilize o RM, informe manualmente um nome para o usuário.

No campo Nome, informe o mesmo nome definido no Usuário Associado ERP.
Em seguida:

  • Defina o perfil do usuário.
  • Selecione as empresas às quais o usuário terá acesso.

Clique em Salvar para concluir o cadastro.

Defina a senha do usuário conforme uma das opções abaixo:

  1. Pedir ao usuário para digitar a mesma senha do RM; ou
  2. Definir uma senha padrão (exemplo: 123456) e orientar o usuário a alterá-la no primeiro acesso.
