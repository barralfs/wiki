Reiniciar NSU

Em alguns casos, o SyncBot pode não retornar todas as notas disponíveis. Para forçar uma nova tentativa de busca, é possível redefinir o NSU (Número Sequencial Único).

Procedimento

  1. Acesse Configurações do Sistema
  2. Abra a aba Scripts
  3. Execute o comando abaixo:

UPDATE CONFIG SET VALOR = '0' WHERE NOME LIKE '%NSU_NFSE%'

Resultado esperado

O SyncBot reiniciará o controle do NSU e realizará uma nova busca das notas disponíveis no próximo ciclo de processamento.

Atenção

Execute este procedimento apenas quando necessário, pois ele reinicia o histórico de controle de buscas.
