Manual Básico – Utilização do Sync XML

1. Introdução

O Sync XML é uma ferramenta utilizada para a integração de documentos fiscais de entrada disponibilizados na SEFAZ com o TOTVS RM.
Seu objetivo é tornar o processo de integração mais intuitivo, ágil e centralizado, evitando que o usuário precise realizar a integração diretamente pelo RM.

2. Login no Sistema

Para acessar o Sync XML, o usuário deve realizar o login no sistema.

⚠️ Importante:
A senha do usuário do Sync XML é a mesma senha utilizada no TOTVS RM.

Após o login, será necessário selecionar a Filial em que o processo será realizado.

3. Manifestação do Destinatário

Após o login, a primeira operação obrigatória é acessar o painel de Manifestação do Destinatário.

  1. Acesse o painel Manifestação do Destinatário.
  2. Clique em “Buscar NFE Sefaz (A partir da última)”.
    • Essa opção busca na SEFAZ todas as notas fiscais a partir da última já consultada.
  3. Ao final da busca, o sistema informará a quantidade de documentos encontrados.

3.1 Filtros de Pesquisa

Utilize os filtros disponíveis (Mês Atual, Mês Anterior, Todos, etc.) e clique em Pesquisar para facilitar a visualização dos documentos.

3.2 Registro da Manifestação

  1. Localize a nota desejada.
  2. Clique com o botão direito do mouse sobre a nota.
  3. Selecione a manifestação desejada (Ciência da Operação, Confirmação da Operação, etc.).

3.3 Download do XML

Após registrar a manifestação:

  1. Clique novamente com o botão direito sobre a nota.
  2. Selecione “Download XML” para baixar o arquivo.
