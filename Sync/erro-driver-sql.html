Erro de Driver de SQL

Ao abrir ou utilizar o Sync, pode ser exibida uma mensagem de erro relacionada ao driver de SQL Server, impedindo a conexão com o banco de dados.

Esse erro ocorre quando o Microsoft ODBC Driver for SQL Server não está instalado ou está desatualizado na máquina onde o Sync está sendo executado.

Como resolver

  1. Acesse a pasta: C:\Sync\Util
  2. Execute o instalador Microsoft ODBC Driver 18 for SQL Server.
  3. Conclua a instalação e reabra o Sync.
